# cpp-sugar

Some code snippets that are often used in c++

## Scripts

```bash
# format code
sh scripts/format.sh

# build
sh scripts/build.sh

# install
sh scripts/install.sh

# run unit test
sh scripts/run_ut.sh

# run unit test and generate codecov report
sh scripts/codecov.sh
```
